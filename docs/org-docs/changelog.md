# Registro de Cambios (Changelog)

Todos los cambios notables en Picura MD serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es/1.0.0/),
y este proyecto sigue los principios de [Versionado Semántico](https://semver.org/spec/v2.0.0.html).

## [0.5.0] - 2024-03-30

### Añadido
- Módulo de interfaz principal con modos de visualización Light/Dark/Auto
- Núcleo del editor Markdown con soporte para sintaxis extendida
- Sistema básico de sincronización con almacenamiento local
- Soporte inicial para integración con GitHub
- Visor de documentos con modo de lectura optimizado
- Asistente de IA local para ayuda contextual
- Dashboard inicial con métricas de sostenibilidad
- Sistema de búsqueda local de documentos
- Componente de navegación por carpetas y etiquetas
- Exportación básica a PDF, HTML y texto plano

### Sostenibilidad
- Implementación de ahorro de batería en modo móvil
- Sistema de monitoreo de consumo de recursos
- Optimización de almacenamiento mediante compresión adaptativa
- Métrica de eficiencia energética en el dashboard

### Técnico
- Arquitectura base de componentes núcleo
- Sistema de extensiones y plugins
- API interna para comunicación entre componentes
- Sistema de gestión de versiones local

## Guía de versionado

- **MAJOR** (X.0.0): Cambios incompatibles con versiones anteriores
- **MINOR** (0.X.0): Nuevas funcionalidades manteniendo compatibilidad
- **PATCH** (0.0.X): Correcciones de errores manteniendo compatibilidad

Versiones por debajo de 1.0.0 se consideran en desarrollo activo y pueden tener cambios significativos entre versiones.
